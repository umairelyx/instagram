# Instagram Profile Clone

A Flutter replica of Instagram's user profile page with clean architecture and video playback.

## Features

- ✅ Profile header with verification badge
- ✅ Profile stats (posts, followers, following)
- ✅ Profile picture with gradient story ring
- ✅ Bio section with links and "Followed by"
- ✅ Action buttons (Following, Message, Add)
- ✅ Story highlights
- ✅ Tab navigation (Posts, Reels, Tagged)
- ✅ Posts grid with tap to view
- ✅ Reels with video playback
- ✅ Tagged photos
- ✅ Bottom navigation bar

## Quick Start

```bash
flutter pub get
flutter run -d chrome
```

## Architecture

Clean Architecture with BLoC/Cubit state management:
- **Domain**: Entities, repositories, use cases
- **Data**: Models, data sources, repository implementations
- **Presentation**: Pages, widgets, state management

## Tech Stack

- Flutter 3.9.2+
- BLoC/Cubit for state management
- Video Player for reel playback
- Clean Architecture pattern
